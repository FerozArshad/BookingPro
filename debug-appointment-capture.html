BSP Appointment Capture Debug Test

This page will automatically run appointment capture tests when loaded.

Open Developer Console (F12) to see results.

Manual Test Functions:

  • bspTestAppointmentAccess() - Check appointment data access
  • bspTestLeadCaptureData() - Test form data collection
  • bspCreateTestAppointments() - Create test appointment data
  • bspTestLeadCapture() - Simulate lead capture
  • bspRunAllTests() - Run all tests

Debug Steps:

  1. Load this page with booking form
  2. Select some appointments in the booking form
  3. Open Developer Console (F12)
  4. Check console output for test results
  5. Run bspTestLeadCapture() to test capture
  6. Check Network tab for AJAX request
  7. Check server debug log for appointment processing
